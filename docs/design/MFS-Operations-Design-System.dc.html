The one hard rule

Components reference semantic tokens only — never a raw hex, never even a primitive. If a primitive leaks into a component, re-theming silently breaks. Treat it as architecture, lint-enforced on new and touched code.

Body & UI — Inter.

Adieu is the brand display face. Where the licensed file isn't loaded, Inter is the defined fallback — so headings degrade gracefully without ever leaving the token system. No more Plus Jakarta Sans wired in outside the tokens.

Patterns · Cross-cutting

Concerns in the seams

App-wide behaviours that are easy to break in a redesign. The system preserves every one — drop any and a real warehouse or food-safety workflow breaks.

PWA & offline

The SyncDot and “my activity today” list are how staff know a tap reached the server. Both survive the redesign. PwaGuard bounces a backgrounded standalone app back to a sensible page.

SyncingStuckClear

PIN paradigm

Sign-in is a 4-digit PIN pad — phone-unlock style, not a password field. A first-class component, used at /login and the /haccp kiosk door.

• • •

Android & Sunmi printing

Ships as a Capacitor Android APK. Label printing is device-native (Sunmi V3) via the two-button PrintLabelStrip — and must behave identically in the webview and the PWA.

100mm AirPrint58mm silent

Leaflet maps

Quarantined behind adapters — the one place the map vendor lives. We restyle the surrounding page and controls and accept limited interior theming (vendor CSS, inline popups, raw-hex pins).

Print · PDF · Excel

Picking lists, jsPDF exports and xlsx spreadsheets have their own layout rules — separate render paths, never styled as web pages.

EN / TR i18n

English + Turkish via t() with a header toggle. All new text routes through t(); layouts assume variable-length strings — Turkish runs longer.
